**Tech Stack:** FastAPI、LangGraph、LangChain、SQLite、Qdrant、Pydantic、Vue 3、TypeScript、pytest、前端 contract test。

## 0. 前置条件、阶段边界与完成定义

前置条件：阶段一～四已经提供版本、工作流、诊断、建议、审核、动作执行和有限智能增强的基础能力，但“已有接口”不等于“业务闭环已验收”。

本阶段交付：

- 以 `workflow_id/run_id` 串联输入文件、输入版本、问题、建议、审核批次、执行记录、输出版本和报告。
- 对全量节点记录轻筛查、候选召回、深诊断、跳过原因和部分完成状态。
- 统一稳定问题代码与中文展示名称，让问题、证据和建议动作可相互追溯。
- 让审核批次具备明确状态机，覆盖逐条、批量、编辑、重新生成、全部拒绝和预览失效。
- 执行前生成不可伪造的动作预览；执行后创建快照、新版本、复验结果和最终报告。
- 前端形成任务中心、审核中心、版本中心、报告中心和数据管理入口。
- 使用固定样例完成后端全量测试、前端 contract/build 和端到端业务验收。

本阶段不做：

- 不建设分布式多 Agent 平台、外部任务队列或完整模型网关。
- 不以增加 LLM 调用次数代替全量规则筛查和覆盖率设计。
- 不允许 LLM 直接写入 Excel、数据库、向量库或绕过审核执行动作。
- 不覆盖原始 Excel；新版本、导出物和报告必须单独保存。
- 不向前端展示原始 chain-of-thought，只展示决策摘要、工具调用和证据。

统一完成定义：

1. service/repository、API/workflow 和前端交互均接入，不存在硬编码演示结果。
2. 成功、失败、拒绝、部分完成、重试和幂等路径有自动化测试。
3. 每个副作用都能提供 workflow/run、版本、审核、预览和执行证据。
4. 固定 Excel 端到端验收通过，并能重复生成结构一致的报告和导出物。
5. 后端全量测试、前端 contract test、typecheck 和 production build 通过。

### Task 4: 建立审核批次状态机与预览失效机制

**目标：** 让审核从“修改建议状态”升级为完整业务状态机，保证只有当前、完整、已通过且可执行的建议能够进入执行。

- [ ] **Step 1: 定义批次和建议状态转换**

批次至少支持 `pending_review/reviewing/ready/executing/executed/failed/superseded`；建议至少支持 `pending/approved/rejected/deferred`。状态转换由 service 校验，API 不直接拼 SQL。

- [ ] **Step 2: 支持逐条、批量和编辑后通过**

批量操作只能影响仍处于 pending 的建议，已完成决定不能被无意覆盖。编辑建议时重新运行一致性检查，并记录原值、修改值、审核决定、备注和时间。

- [ ] **Step 3: 处理特殊业务分支**

- 全部拒绝：允许正常结束审核，但拒绝创建空执行预览。
- 零问题/零建议：允许生成空诊断结论并完成，不制造演示问题。
- 建议不完整：允许原位重新生成或转人工，不重复创建问题。
- 无可执行动作：报告说明原因，不能创建空新版本。

- [ ] **Step 4: 预览 hash 与失效**

预览绑定 review hash、输入版本和建议集合。任一审核决定或建议内容变化后，旧预览立即失效；即使数据库状态被错误改成 ready，执行 service 仍需重新校验 hash。

## 4. 风险与降级策略

- **本地工作树改动较多：** 每个 Task 开工前检查 `git status` 和相关文件 diff，只做最小重叠修改；不得回退无关改动。
- **SQLite 迁移风险：** 迁移必须可重复运行，先检查字段和索引是否存在；不得以重建数据库替代兼容性处理。
- **模型和 Qdrant 不可用：** 规则诊断、人工审核、版本管理和报告主路径仍应可用；外部增强标记为降级，不伪造成功。
- **报告事实漂移：** 页面、Markdown/PDF 和 Excel 统一消费同一聚合 DTO；禁止多处独立计算评分和数量。
- **状态机复杂度：** 状态转换集中在 service，并用表驱动测试覆盖非法转换；API 和 Vue 组件只消费 capability。
- **副作用恢复：** 数据库事务、文件临时产物、快照、幂等键和操作日志缺一不可；无法保证回滚的动作不得进入自动执行。
- **大文件性能：** 阶段验收至少记录固定大样例的解析、诊断、报告与导出耗时；未达到目标时明确标为待优化，不隐藏超时。
